CSS classes are used to select the HTML elements we want to style. Classes are declared in the HTML element's class attribute. To select an element with a specific class, write a dot (.) character, followed by the class we declare in the HTML element's class attribute.

CSS Id selectors are used to select the element with a unique id. To select an element with a specific id, write a hash (#) character, followed by the id we declare in the HTML element's id attribute. Ids are stronger than classes. If an element has both an id and a class, the properties of the id will override the properties of the class.

CSS element selectors are used to select HTML elements based on the element name. To select an element with a specific element name, write the element name. Be careful when using element selectors, as they select all elements of that type. For example, if you use the element selector to select the h2 element, all h2 elements on the page will be selected.

Inline Styles

Inline styles are used to style directly on the HTML element. This has the highest priority, and will override any other styling.
